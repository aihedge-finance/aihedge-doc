# Welcome to AIHedge Documentation

**AIHedge Finance** is an AI-powered, non-custodial decentralized hedge fund protocol built on the **Yearn V3 ERC-4626** vault architecture.

This documentation covers everything you need to understand and interact with the AIHedge protocol — from core concepts and vault mechanics to smart contract references and developer guides.

---

## What is AIHedge?

AIHedge bridges the gap between passive capital allocators and sophisticated, machine-learning-driven execution engines. It enables:

- **Yield Vaults** — Multi-strategy ERC-4626 vaults that automatically allocate, rebalance, and compound yield across DeFi protocols
- **Vault Creation** — A permissionless system for curators to deploy custom yield strategies
- **AI Optimization** — Reinforcement Learning, CNN pattern recognition, and Bayesian tuning to maximize risk-adjusted returns
- **Self-Custody** — Non-custodial by design: your assets remain in audited, immutable on-chain smart contracts
