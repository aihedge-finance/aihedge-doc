# Supported Networks & Wallets

The AIHedge protocol and DApp interface support the following blockchain networks and Web3 wallets.

## Supported Networks

You can deposit assets and run strategies on these networks:

* **Ethereum Mainnet (ETH Main)**
* **BNB Chain**
* **Arbitrum**
* **Base**
* **Optimism**

## Supported Wallets

Connect your secure Web3 wallet to interact with the platform:

* **WalletConnect**
* **OKX Wallet** (Browser)
* **MetaMask** (Browser)
